Grid 기본 개념

컨테이너에 display: grid를 선언하면 자식 요소가 Grid Item이 됩니다

핵심 용어
  • Grid Container — display: grid를 선언한 부모 요소
  • Grid Item — 컨테이너의 직접 자식 요소들
  • Grid Line — 행·열을 나누는 가상의 선 (번호로 참조)
  • Grid Track — 두 Grid Line 사이의 행 또는 열
  • Grid Cell — 행·열이 교차하는 단일 칸
  • Grid Area — 하나 이상의 Cell을 합친 영역

grid-template-columns 예시

열의 크기와 개수를 다양한 방법으로 정의할 수 있습니다

css 
/* 200px 고정 + 나머지 균등 */
.layout {
  display: grid;
  grid-template-columns:
    200px 1fr 1fr;
}

/* repeat() 단축 표기 */
.grid-4 {
  grid-template-columns:
    repeat(4, 1fr);

grid-area 활용

이름을 붙여 레이아웃 영역을 직관적으로 배치합니다

css 
.container {
  display: grid;
  grid-template-columns:
    1fr 3fr 1fr;
  grid-template-rows:
    auto 1fr auto;
  grid-template-areas:
    "header  header  header"
    "sidebar main    aside"
    "footer  footer  footer";
}

.header  { grid-area: header;  }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main;    }
.aside   { grid-area: aside;   }
.footer  { grid-area: footer;  }
