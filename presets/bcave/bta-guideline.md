# BTA 컨셉화 운영 지침 (BCAVE)

> 출처: BTA 컨셉 보고 (1차정리) — 브랜드사업본부 전략기획실, 2025.11.21

---

## 1. BTA란?

**B**asic · **T**rend · **A**ccent — 브랜드의 아이덴티티 강화, 감도 개선, 매출 극대화 등 서로 상반되는 목표를 동시에 달성하기 위한 **상품과 컬러 구성전략**.

### 핵심 요약

| 구분 | 목적 | 주요 포인트 |
|------|------|----------|
| **Basic** | 안정적 매출 | 신뢰 / 품질 / 지속성 |
| **Trend** | 시장감도 강화 | 브랜드화된 트렌드 |
| **Accent** | 미래 매출 자원 | 실험 / 아이덴티티 강화 |

### BTA는 브랜드 포트폴리오 관리 시스템이다

- 기획, 소싱, 디자인, 마케팅, 영업, VM 협의체를 통해 **공통의 언어**로 설계해야 브랜드 메시지가 일관된다.
- 워크플로우 프로세스를 수립하기 위해 **통합시트**를 만드는 것을 제안한다.

---

## 2. BTA 결정요소

| 구분 | 내용 |
|------|------|
| **정량 지표** | 자사/타사 판매, 정상 판매율, 할인율, 재구매율, 검색 및 관심도 등 |
| **현재 트렌드** | 시즌/내수 업계 동향, SNS, 해외 레퍼런스 브랜드 등 |
| **미래 예측 자료** | 트렌드 리포트, 선행지표(소셜/매거진 등), 컬러 포캐스트, 소재 트렌드, 문화 소비 키워드, 내부 아이디에이션 등 |

### BTA 구분 및 표현 방법

- 상품선정과 디자인 4요소(**컬러, 실루엣, 소재, 디테일**)를 통해 BTA 구분 및 표현
- 상품 부서뿐 아니라 영업, 마케팅 컨텐츠 등과 관여되므로 영업, 마케팅, VM 간의 **사전 합의 및 조율** 필수

### BTA 적용 범위

기획 · 디자인 · 소싱 · 마케팅 · 영업 · VM **전반**

## 3. B (Basic) — 고객 접점이 가장 큰 신뢰의 영역

> 안정적 매출 확보를 위한 상품으로, 고객 접근성이 좋고 재구매율이 높은 핵심 라인

### 정의

| 항목 | 내용 |
|------|------|
| **목적** | 안정적 매출 확보 |
| **기반** | 과거~현재의 자사/타사 정량적 성과 기반 |
| **상품 특징** | 접근성이 좋고, 재구매율이 높은 핵심 라인 |
| **디자인 방향** | 상품 속성상 소비자가 지루함을 느낄 수 있으므로 **신선도 상향을 위한 지속적 업그레이드** 필요 (소재, 컬러, 그래픽, 실루엣 등) |
| **컬러 전략** | 베이직 컬러 기본, 신선도 유지를 위해 트렌드/악센트 컬러 배치하여 단조로움 탈피 |
| **기획 전략** | 전 카테고리 최소 1스타일 이상 필수 배치, 스타일수 대비 절대 수량이 큰 영역 |
| **소싱 전략** | 사전기획, 대량생산, 안정적 벤더, 안정적 소재, 원가 효율 중심 |
| **마케팅 연동** | 신뢰, 리텐션 중심 스토리텔링 |
| **KPI** | 회전율, 수익률, 재구매율 |

### 부서별 액션

| 부서 | 액션 |
|------|------|
| **D실 (디자인)** | 베이직 특성상 지루함 방지 → 소재·컬러·그래픽·실루엣 등 작년과 달라지는 요소 반드시 포함. 베이직 컬러 기본 + 트렌드/악센트 컬러 구성으로 단조로움 탈피 (컬러 BTA) |
| **상품기획** | 전 카테고리 최소 1스타일 이상 필수 배치, 스타일 수 대비 발주 수량이 큰 상품 |
| **생산** | 선기획(비수기), 대량 생산, 안정적 벤더, 안정적 소재, 원가 효율 중심 |
| **마케팅** | 신뢰, 리텐션 중심 스토리텔링 |

### 주의점

- **퀄리티 결정 영역**이므로 완성도 높은 봉제, 소재, 핏 등 품질이 가장 좋아야 되는 영역
- 익숙함으로 인한 **상품 피로도 방지**를 위한 지속적인 디자인 업데이트 필요

## 4. T (Trend) — 매출과 감도를 동시에 견인하는 중심 영역

> 매출 확대, 브랜드 감도를 높이는 상품으로, 현재 시장의 유행을 브랜드 관점(정체성)에서 해석한 상품군

### 정의

| 항목 | 내용 |
|------|------|
| **목적** | 매출 확대, 감도 상향 |
| **기반** | 정량적/정성적 접근 모두 필요. 트렌드=유행. 즉 **최근 1~2년 유행했고, 향후 1~2년 확장 가능한 트렌드** |
| **상품 특징** | 현재 시장의 유행을 **브랜드 관점(정체성)**에서 해석한 상품군 |
| **디자인 방향** | 시장 트렌드를 반영하되 브랜드 정체성을 유지 |
| **컬러 전략** | 베이직+트렌드+악센트 컬러 모두 활용하되 카테고리별 선택적 배치 |
| **기획 전략** | 유행의 한가운데에 있는 상품 → 기획/디자인실에서 가장 공들여야 하는 상품군. 브랜드 인지도와 매출을 견인하는 영역으로 **스타일수를 가장 많이 배치** |
| **소싱 전략** | 단기/중기 리드 타임 확보, 유연한 리오더 구조, 트렌드 소재 대응 |
| **마케팅 연동** | 시즌 캠페인의 핵심 노출군, 룩북, SNS 주력 사용군 |
| **KPI** | 매출 비중, 회전율, SNS 반응도, 캠페인 노출 효과 등 |

### 부서별 액션

| 부서 | 액션 |
|------|------|
| **D실 (디자인)** | 시장 트렌드를 반영하되 브랜드 정체성을 유지. 베이직+트렌드+악센트 컬러 모두 활용하되 카테고리별 선택적 배치 |
| **상품기획** | 유행의 한가운데에 있는 상품 → 기획/디자인실에서 가장 공들여야 하는 상품군. 브랜드 인지도와 매출을 견인하는 영역으로 스타일수를 가장 많이 배치 |
| **생산** | 단기/중기 리드 타임 확보, 유연한 리오더 구조, 트렌드 소재 대응 |
| **마케팅** | 시즌 캠페인의 핵심 노출군, 룩북, SNS 주력 사용군 |

### 주의점

- 트렌드를 **'복제'가 아닌 '브랜드화'** 하는 해석력 필요

## 5. A (Accent) — 감도와 미래 매출 자원 발굴의 실험 영역

> 브랜드 감도 상향, 신규 트렌드 발굴을 위한 상품으로, 시즌을 입체감있게 표현하는 '아이캐칭' 상품

### 정의

| 항목 | 내용 |
|------|------|
| **목적** | 감도 상향, 신규 상품 발굴 |
| **기반** | 트렌드 리포트, 글로벌 레퍼런스 자료 등 예측 기반 |
| **상품 특징** | 시즌을 입체감있게 표현하는 **'아이캐칭' 상품** |
| **디자인 방향** | 챌린지와 차별화된 디테일로 **감도 중심 디자인** |
| **컬러 전략** | 상품 자체가 악센트이므로 밸런스를 위해 **베이직/트렌드 컬러**를 적용 |
| **기획 전략** | 매출 비중 높은 카테고리 또는 강화 카테고리 내의 상품 선정. 수량 기획은 **보수적**으로 집행 |
| **소싱 전략** | 단기생산, 소량 생산 가능 업체, 실험적 소재 풀 넓은 업체 활용 |
| **마케팅 연동** | 비주얼, PR, 룩북 등에 환기성 이미지용 (매출보다는 **감도 중심**) |
| **KPI** | 이미지 기여도, 차시즌 트렌드 발굴비율(전이율) |

### 부서별 액션

| 부서 | 액션 |
|------|------|
| **D실 (디자인)** | 챌린지와 차별화된 디테일로 감도 중심 디자인. 상품 자체가 악센트이므로 밸런스를 위해 베이직/트렌드 컬러를 적용 |
| **상품기획** | 매출 비중 높은 카테고리 또는 강화 카테고리 내의 스타일 선정. 수량 의사결정은 보수적으로 |
| **생산** | 단기생산, 소량 생산 가능 업체, 실험적 소재 풀 넓은 업체 활용 |
| **마케팅** | 비주얼, PR, 룩북 등에 환기성 이미지용 (매출보다는 감도 중심) |

### 주의점

- 챌린지(실험) 상품이더라도 **'브랜드 정체성'과의 연계가 필수**

## 6. BTA 컬러 운용 원칙

### 기본 원칙

1. 컬러 역시 매 시즌 상품 BTA별 전략과 동일하게 **BTA컬러를 선정 및 구분하여 운영**
2. 악센트 컬러는 기본적으로 **베이직 상품에 적용** (예외 가능하나 시즌 전체 맵에서 밸런스 점검 후 허용)
3. 악센트 상품은 상품 자체가 악센트이므로 **디자인 균형을 위해 베이직 또는 트렌드 컬러**를 적용

### 브랜드 룩북 촬영 시 원칙

| BTA 구분 | 룩북 컬러 원칙 |
|---------|-------------|
| **Basic 상품** | 악센트 컬러 중심 |
| **Trend 상품** | 컬러 무관 |
| **Accent 상품** | 필수 노출 |

## 7. 부문별 역할 및 협업 구조

| 부문 | 주요 역할 | BTA 협업 포인트 |
|------|---------|--------------|
| **상품기획(MD)** | 카테고리별 BTA 비율 설정, 수량·가격 조정 | 판매데이터 및 리스크 조정 역할 |
| **디자인** | 상품의 감도 조율, BTA별 디자인 가이드 구축 | 디자인 방향성과 감도 밸런스 주도 |
| **마케팅(IMC)** | 시즌별 BTA 노출 전략 및 메시지 설계 | 캠페인 스토리텔링 구조화 |
| **소싱/생산** | 원가·리드타임 관리, BTA별 생산체계 구분 | 효율·리스크 균형 조정 |
| **영업/VM** | 매장별 BTA 재고 구성 비율 관리 | 소비자 피드백 수집 및 회전율 피드백 |

### IMC와의 연계

- 전략실에서 IMC 기준 수립 후 연동하여 연계 기준 수립 필요

### 운영 프로세스 (시즌 워크 플로우)

- 실무단에서 수립 필요 (디벨롭 예정)

## 8. 주의사항

1. **BTA에 대한 이해도 높이기**: 3개 브랜드 공통으로 하는 레퍼런스 브랜드들의 이미지로 내부 컨셉 공유
   - 예: 사업부별 레퍼런스 브랜드가 정리되고, 전 직원의 머리속에 있어야 함
   - (ex. 00사업부: 폴로, 스투시, 슈프림, 나나미카 등)
2. **상품 부서뿐 아니라** 영업, 마케팅 컨텐츠 등과 관여되므로 영업, 마케팅, VM 간의 사전 합의 및 조율 필수

## 9. FPOF 연동 가이드

BTA 전략은 FPOF의 다음 스킬/단계와 직접 연동됩니다:

| FPOF 스킬 | BTA 연동 포인트 |
|----------|-------------|
| `md-planning` | 카테고리별 BTA 비율 설정, SKU 배분 |
| `line-sheet` | BTA별 물량 배치, 사이즈 비율 |
| `brand-strategy` | BTA를 활용한 브랜드 포지셔닝 정의 |
| `moodboard` | BTA별 디자인 무드 구분 |
| `design-spec` | BTA 구분에 따른 디자인 4요소 스펙 |
| `costing-ve` | BTA별 원가 구조 차별화 (B=효율, T=유연, A=실험) |
| `imc-strategy` | BTA별 마케팅 노출 전략 및 메시지 |
| `visual-content` | 룩북 촬영 시 BTA 컬러 운용 원칙 적용 |
| `sales-analysis` | BTA별 KPI 분석 (B=회전율/재구매, T=매출비중/SNS, A=이미지기여도) |

### BTA 기반 시즌 기획 체크리스트

- [ ] 시즌 BTA 비율 설정 완료 (MD)
- [ ] BTA별 레퍼런스 브랜드 정리 완료
- [ ] BTA별 컬러 팔레트 선정 완료 (디자인)
- [ ] BTA별 소싱 전략 수립 완료 (생산)
- [ ] BTA별 마케팅 메시지 설계 완료 (IMC)
- [ ] 부서간 BTA 기준 사전 합의 완료
- [ ] 통합시트 작성 완료
