# 좋은 예시 (Good Example)

아래는 이 스킬로 생성된 **모범적인 요약 보고서** 예시입니다.

---

## 예시: 주간 마케팅 성과 보고서

```markdown
# 2026년 1월 2주차 마케팅 성과 보고

> **작성일:** 2026-01-15
> **작성자:** 마케팅팀
> **보고 대상:** 주간 경영회의

---

## Summary

1. 1월 신규 캠페인 "겨울 할인전"이 1/8 런칭되어, 첫 주 CTR 4.2%로 목표(3.5%) 대비 120% 달성했습니다.
2. 전환율(CVR)은 2.1%로 전주 대비 +0.3%p 상승했으며, 특히 모바일 채널에서 +0.5%p 개선이 두드러졌습니다.
3. 다음 주에는 A/B 테스트 결과를 반영하여 광고 소재를 최적화하고, 리타겟팅 캠페인을 확대할 예정입니다.

---

## Key Metrics

| 지표 | 이번 주 | 전주 대비 | 목표 대비 | 비고 |
|------|---------|----------|----------|------|
| CTR (클릭률) | 4.2% | +0.8%p | 120% | 역대 최고 |
| CVR (전환율) | 2.1% | +0.3%p | 105% | 모바일 ↑ |
| CPA (획득비용) | 12,500원 | -1,200원 | 96% | 효율 개선 |
| ROAS (광고수익률) | 320% | +45%p | 107% | - |
| 신규 가입자 | 1,847명 | +23% | 92% | 목표 미달 |

---

## Insights

### 성과 요인
- **CTR 상승**: 새 광고 소재(동영상)의 클릭 유도 효과가 검증되었습니다. 특히 15초 숏폼 영상이 정적 이미지 대비 CTR +1.2%p 높았습니다.
- **CPA 절감**: 오디언스 세그먼트 재설정으로 비효율 타겟이 제거되어 획득 비용이 감소했습니다.

### 개선 필요 사항
- **신규 가입자 목표 미달(92%)**: 가입 전환 퍼널 분석 결과, 회원가입 폼에서 이탈률이 35%로 높았습니다. UX 개선이 필요합니다.
- **데스크톱 CVR 정체**: 모바일 대비 데스크톱 전환율이 0.8%p 낮아, 랜딩페이지 최적화가 필요합니다.

### 시사점
- 동영상 소재 투자 확대 시 추가 성과 기대됩니다.
- 회원가입 UX 개선은 전환율 +0.5%p 이상 개선 잠재력이 있습니다.
```

## 왜 이 예시가 좋은가?

### ✅ 체크리스트 평가

| 항목 | 충족 여부 | 설명 |
|------|----------|------|
| Summary 3줄 이내 | ✅ | 목적-결과-다음단계 구조 |
| 수치가 표로 정리됨 | ✅ | 5개 핵심 지표, 전주/목표 대비 포함 |
| Insights에 "왜" 포함 | ✅ | 성과 요인과 원인 분석 명시 |
| Action Item에 담당자 | ✅ | 모든 항목에 담당자 지정 |
| Action Item에 마감일 | ✅ | 구체적 날짜(YYYY-MM-DD) |
| 경어체 사용 | ✅ | "~입니다", "~했습니다" |
| 한영 병기 | ✅ | CTR(클릭률), CVR(전환율) 등 |

### 💡 핵심 포인트

1. **압축률**: 원본 데이터 대비 약 80% 압축
2. **실행 가능성**: 모든 Action Item이 구체적이고 추적 가능
3. **의사결정 지원**: Insights가 다음 액션의 근거를 제공

## 나쁜 예시와 비교

### ❌ 나쁜 예시의 특징

```markdown
# 마케팅 보고

이번 주 캠페인이 잘 되었습니다. 클릭률이 올랐고 전환도 좋아졌습니다.
모바일이 특히 좋았습니다. 다음 주에는 더 열심히 하겠습니다.
회원가입은 조금 부족했는데 개선하겠습니다.

할 일:
- 소재 만들기
- 테스트 하기
- 보고서 쓰기
```

### 문제점

| 문제 | 설명 |
|------|------|
| 수치 없음 | "좋아졌다"만 있고 구체적 수치 없음 |
| 구조 없음 | 섹션 구분 없이 나열 |
| 담당자/기한 없음 | 추적 불가능한 Action Item |
| 모호한 표현 | "더 열심히", "조금 부족" 등 |
