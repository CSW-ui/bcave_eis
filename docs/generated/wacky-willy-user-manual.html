2장 — 기본 사용법

자연어로 말하는 법

✅ 잘하는 방법
"그래픽 티 무드보드 만들어줘"
"오버핏 후디 디자인 스펙 작성해줘"
"이번 시즌 트렌드 분석해줘"
❌ 피해야 할 방법
"알아서 해" → 너무 모호함
"그거 만들어" → 무엇인지 모름
"뭐 할까?" → 스스로 결정 안 함

Tip: 구체적으로 말할수록 좋습니다 — "키키 캐릭터를 활용한 유니섹스 그래픽 티셔츠 디자인해줘. 타겟은 18~22세, 스트리트 감성으로"

6장 — FAQ

자주 묻는 질문

브랜드와 안 맞는 결과?

더 구체적인 프롬프트 사용. "와키윌리의 Kitsch Street 감성으로 무드보드 만들어줘"

단계를 건너뛰고 싶어요

"강제 통과해줘"로 가능하지만 권장하지 않음. 빠진 부분 직접 보완 필요

산출물이 마음에 안 들어요

바로 수정 요청. "여기 수정해줘: 컬러를 Blue로 바꿔" 또는 "다시 만들어줘"

QG에서 계속 FAIL

누락된 항목 확인 → 해당 항목 생성 요청 → 재검수
