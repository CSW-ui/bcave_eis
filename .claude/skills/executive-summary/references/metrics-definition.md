# 지표 정의 사전 (Metrics Definition)

업무 요약 보고서에서 자주 사용되는 지표들의 정의와 계산법을 정리합니다.

---

## 마케팅 지표 (Marketing Metrics)

### 트래픽 & 참여 (Traffic & Engagement)

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **노출수** | Impressions | 광고/콘텐츠가 화면에 표시된 횟수 | - |
| **클릭수** | Clicks | 광고/콘텐츠를 클릭한 횟수 | - |
| **클릭률** | CTR (Click-Through Rate) | 노출 대비 클릭 비율 | `(클릭수 / 노출수) × 100` |
| **이탈률** | Bounce Rate | 단일 페이지만 보고 이탈한 비율 | `(이탈 세션 / 전체 세션) × 100` |
| **페이지뷰** | PV (Page Views) | 페이지가 조회된 총 횟수 | - |
| **순방문자** | UV (Unique Visitors) | 중복 제외 방문자 수 | - |
| **체류시간** | Session Duration | 평균 사이트 체류 시간 | `총 체류시간 / 세션 수` |

### 전환 (Conversion)

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **전환율** | CVR (Conversion Rate) | 방문자 대비 전환 비율 | `(전환 수 / 방문자 수) × 100` |
| **전환수** | Conversions | 목표 행동 완료 횟수 | - |
| **리드** | Leads | 잠재 고객 수 | - |
| **가입률** | Sign-up Rate | 방문자 대비 가입 비율 | `(가입 수 / 방문자 수) × 100` |

### 비용 효율 (Cost Efficiency)

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **클릭당비용** | CPC (Cost Per Click) | 클릭 1회당 비용 | `광고비 / 클릭수` |
| **노출당비용** | CPM (Cost Per Mille) | 1,000회 노출당 비용 | `(광고비 / 노출수) × 1000` |
| **획득비용** | CPA (Cost Per Acquisition) | 전환 1건당 비용 | `광고비 / 전환수` |
| **리드당비용** | CPL (Cost Per Lead) | 리드 1건당 비용 | `광고비 / 리드수` |
| **설치당비용** | CPI (Cost Per Install) | 앱 설치 1건당 비용 | `광고비 / 설치수` |

### 수익 & ROI (Revenue & ROI)

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **광고수익률** | ROAS (Return On Ad Spend) | 광고비 대비 매출 | `(광고 매출 / 광고비) × 100` |
| **투자수익률** | ROI (Return On Investment) | 투자 대비 순이익 | `((수익 - 비용) / 비용) × 100` |
| **객단가** | AOV (Average Order Value) | 주문당 평균 금액 | `총 매출 / 주문 수` |
| **고객생애가치** | LTV (Lifetime Value) | 고객 1인당 총 기대 매출 | `객단가 × 평균 구매 횟수 × 평균 유지 기간` |
| **고객획득비용** | CAC (Customer Acquisition Cost) | 고객 1인 획득 비용 | `총 마케팅비 / 신규 고객 수` |

## 프로젝트 관리 지표 (Project Management Metrics)

### 진행률 & 일정

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **완료율** | Completion Rate | 전체 대비 완료 비율 | `(완료 태스크 / 전체 태스크) × 100` |
| **일정 준수율** | On-Time Rate | 기한 내 완료 비율 | `(기한 내 완료 / 전체 완료) × 100` |
| **지연일수** | Delay Days | 예정일 대비 지연 | `실제 완료일 - 예정 완료일` |
| **번다운** | Burndown | 남은 작업량 추이 | - |

### 품질 & 효율

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **결함률** | Defect Rate | 전체 대비 결함 비율 | `(결함 수 / 총 항목) × 100` |
| **재작업률** | Rework Rate | 재작업이 필요한 비율 | `(재작업 항목 / 완료 항목) × 100` |
| **생산성** | Velocity | 단위 기간당 완료량 | `완료 스토리포인트 / 스프린트` |

## 이커머스 지표 (E-commerce Metrics)

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **GMV** | Gross Merchandise Value | 총 거래액 | `판매가 × 판매량` |
| **순매출** | Net Revenue | 순수 매출 | `GMV - 할인 - 환불` |
| **장바구니 전환율** | Cart Conversion | 장바구니 담기 후 구매 비율 | `(구매 수 / 장바구니 수) × 100` |
| **이탈률** | Cart Abandonment | 장바구니 이탈 비율 | `((장바구니 - 구매) / 장바구니) × 100` |
| **재구매율** | Repeat Purchase Rate | 재구매 고객 비율 | `(재구매 고객 / 전체 고객) × 100` |

## 고객 지표 (Customer Metrics)

| 지표 | 영문명 | 정의 | 계산식 |
|------|--------|------|--------|
| **NPS** | Net Promoter Score | 추천 의향 점수 | `추천자% - 비추천자%` |
| **CSAT** | Customer Satisfaction | 고객 만족도 | 설문 평균 점수 |
| **이탈률** | Churn Rate | 고객 이탈 비율 | `(이탈 고객 / 기초 고객) × 100` |
| **유지율** | Retention Rate | 고객 유지 비율 | `100% - 이탈률` |
| **DAU** | Daily Active Users | 일일 활성 사용자 | - |
| **MAU** | Monthly Active Users | 월간 활성 사용자 | - |
| **WAU** | Weekly Active Users | 주간 활성 사용자 | - |

## 비교 표현 가이드

보고서에서 변화를 표현할 때 사용하는 표기법:

| 표현 | 의미 | 예시 |
|------|------|------|
| `+X%` | X% 증가 (비율) | CTR +15% |
| `-X%` | X% 감소 (비율) | 이탈률 -5% |
| `+X%p` | X%포인트 증가 | CVR +0.5%p (2.0%→2.5%) |
| `-X%p` | X%포인트 감소 | 이탈률 -2%p |
| `YoY` | 전년 동기 대비 | YoY +20% |
| `MoM` | 전월 대비 | MoM -3% |
| `WoW` | 전주 대비 | WoW +8% |
| `QoQ` | 전분기 대비 | QoQ +12% |
