grid-template grid-area fr 단위 명시적 / 암묵적 그리드 반응형 레이아웃

Grid 기본 개념

CSS Grid는 행(row)과 열(column)을 동시에 제어하는 2차원 레이아웃 시스템입니다.
Flexbox가 1차원(한 방향)이라면, Grid는 표처럼 전체 구조를 한 번에 설계합니다.

◽

Grid Container

display: grid를 선언한 부모 요소. 이 안에 있는 자식이 Grid Item이 됩니다.

◾

Grid Item

컨테이너의 직접 자식 요소. 행·열 교차점인 셀(cell)에 배치됩니다.

☰

Grid Line / Track

셀을 구분하는 선이 Grid Line, 두 선 사이 공간이 Track입니다.

.container {
  display: grid;                          /* Grid 활성화 */
  grid-template-columns: 1fr 1fr 1fr;  /* 3열 균등 분할 */
  grid-template-rows: 100px 200px;     /* 2행 높이 지정 */
  gap: 16px;                           /* 셀 간격 */
}

grid-template-columns 예시

repeat() + fr 단위

/* 균등 3열 */
grid-template-columns:
  repeat(3, 1fr);

/* 2:1 비율 */
grid-template-columns:
  2fr 1fr;

/* auto-fill 반응형 */
grid-template-columns:
  repeat(auto-fill,
    minmax(180px, 1fr));

/* span으로 셀 병합 */
.item-wide {
  grid-column: span 2;
}

fr 단위란?

사용 가능한 공간을 비율로 나누는 단위.
1fr 2fr = 전체를 3등분 후 1 : 2 배분

grid-area 활용

이름을 붙여 레이아웃을 지도처럼 설계합니다.
복잡한 좌표 대신 grid-template-areas로 직관적으로 배치하세요.

.layout {
  display: grid;
  grid-template-columns: 160px 1fr;
  grid-template-rows:
    40px 1fr 40px;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  gap: 6px;
}

header  { grid-area: header;  }
aside   { grid-area: sidebar; }
main    { grid-area: main;    }
footer  { grid-area: footer;  }

렌더링 결과

header
sidebar
main content
footer

팁: 같은 이름을 나란히 쓰면 자동 병합.
"header header" → 헤더가 2열을 차지

Summary

오늘 배운 것 정리

1
Grid Container / Item — display: grid로 활성화, 직접 자식이 Item이 됩니다.
2
grid-template-columns — fr, repeat(), minmax()로 열 구조를 설계합니다.
3
grid-column / grid-row — 라인 번호나 span으로 셀을 병합할 수 있습니다.
4
grid-area + grid-template-areas — 이름 기반 배치로 전체 레이아웃을 지도처럼 선언합니다.
5
auto-fill + minmax() — 반응형 그리드를 미디어쿼리 없이 한 줄로 구현합니다.
